Este projeto consiste em uma tabela dinâmica desenvolvida com HTML, CSS e JavaScript, permitindo ao usuário gerenciar dados de forma prática e intuitiva. A tabela possibilita adicionar novas linhas, editar conteúdos existentes e excluir registros, tudo diretamente na interface.
